# Security Policy

## Supported Versions

We are iterating fast. Only the latest version is actively supported.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you find a security vulnerability, do NOT open a public issue. Please report it privately via [GitHub Security Advisories](https://github.com/lqdflying/OAIProxy/security/advisories).

We aim to respond within 48 hours. Keep it simple: just provide the reproduction steps and the potential impact.
